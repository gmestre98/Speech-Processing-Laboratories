Lab 1

Deadline: March 15th, 2020

  1. Recordings (individual)
    All recordings should be done at 16kHz, linear 16 bits per sample, mono.
    Record all the (oral) vowels of European Portuguese (non-native students should try this), with silences between them. The corresponding file should be named vowels_nnnnn.wav, where nnnnn is the student number.
  2. Vowel triangle (individual) Draw the vowel triangle in an image file entitled vtri_nnnnn.png, including a legend, specifying the vowel. The triangle should have F1 and F2 in the horizontal and vertical axes, respectively. Observe the coincidence or dispersion of the values obtained for the two voices of the group.
  3. Orthographic annotation (individual)
    Do the manual orthographic annotation of the recording corresponding to your birth date (see lab 0) for each of the members of the group, using the Praat or Wavesurfer tools. For Wavesurfer, store the annotation in file birthdate_nnnnn.ort, with a word in each line. For Praat, the orthographic annotation will be one of the tiers of the TextGrid.
  4. Narrow phonetic annotation (individual)
    Do the manual phonetic annotation of the same recording. Use only SAMPA symbols for European Portuguese. For Wavesurfer, store the annotation in file birthdate_nnnnn.lab, with a phone in each line. For Praat, the narrow phonetic annotation will be another tiers in TextGrid.
  5. Broad phonetic transcription (group)
    Select a sentence (or set of sentences) from this week's national newspapers, of 30-50 words. The sentence should not include foreign words, acronyms, or digits. (Indicate the newspaper source, and date.)
    Write the phonetic transcription of each word (citation form), using SAMPA symbols, and assuming a Lisbon accent.
    Write its broad phonetic transcription.
    In this broad transcription, which are the 3 most frequent symbols? And the 3 rarest ones (or non-existent)?
    Store your answers in file sentence.txt.
  6. Visual identification of digit sequences (group)
    Identify the 10-digit sequence assigned to each group by inspecting the corresponding spectrograms and waveforms. There are no repeated digits. The recordings correspond to telephone speech. The last digit of the group number (i) defines the sequence gi to be identified, which is shown in two screens:
    • gia.png - spectrogram and waveform of the first 5 digits;
    • gib.png - spectrogram and waveform of the last 5 digits;
    The answer should be given as an orthographic transcription file (dig_gi.txt) Short comments are welcome.

    • g1a.png and g1b.png
